Suivi des données hydroclimatiques
Objet

Cette section est un tableau de bord intéractif des données hydroclimatiques utilisées pour identifier les événements passés proches de l'événement en cours.

Informations

Le modèle s'appuie sur les données climatiques, hydrométriques et piézométriques pour établir les tendances saisonnières de débits. Un visualiseur de ces données est inclus dans l'outil CYDRE.

  • Les graphiques sont représentés sous forme d'hydrographe saisonnier sur une année civile, en calculant pour chaque jour les quantiles 10, 50 et 90.
  • Le quantile 50 est représenté par la courbe noire tandis que la bande bleue indique la variabilité interannuelle
  • Vous pouvez comparer les courbes d'une ou plusieurs années par rapport à ces quantiles en les sélectionnant dans la liste déroulante.
  • Pour chaque graphique, la dernière date de disponibilité des données, la période de calcul des quantiles et leur source sont indiquées.
  • Pour obtenir les données, cliquez sur le bouton de téléchargement.
