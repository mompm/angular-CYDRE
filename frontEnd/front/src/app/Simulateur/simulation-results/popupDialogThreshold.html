Ajout/Modification de seuils pour le calcul des indicateurs
Objet

Cette section vous permet d'ajouter ou de modifier des seuils supplémentaires, afin de calculer les indicateurs opérationnels. Par défaut le seuil du débit réservé (1/10 du module) est ajouté et ne peut être modifié.

Mode opératoire
  • Ajouter un nouveau seuil : Cliquez sur "+" pour ajouter un nouveau seuil.
  • Définir les paramètres du seuil : Spécifiez la couleur d'affichage, le nom et la valeur en m³/s du seuil. Ce seuil sera représenté par une ligne horizontale sur le graphique des tendances saisonnières de débits.
  • Mettre à jour les indicateurs opérationnels : Cliquez sur "Actualiser les indicateurs". Les indicateurs opérationnels seront recalculés pour chacun des seuils ajoutés dans la section suivante.
